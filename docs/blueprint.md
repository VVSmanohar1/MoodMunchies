# **App Name**: Mood Munchies

## Core Features:

- Mood Input: Allow users to input their current mood either through voice or by selecting from a predefined list.
- Recommendation Engine: Generate personalized food and restaurant recommendations based on user's mood, occasion, cuisine preferences, dietary requirements (veg/non-veg), preferred time, and location.
- Cuisine Filtering: Enable users to filter restaurant recommendations by cuisine type (e.g., Italian, Mexican, Indian) to narrow down options.
- Dietary Preference Filtering: Allow users to filter recommendations based on dietary preferences such as vegetarian, non-vegetarian, vegan, gluten-free.
- Additional Notes Processing: Process any additional notes the user provides via text input to refine recommendations further.

## Style Guidelines:

- Primary color: Soft terracotta (#E07A5F) to create a warm, inviting feel.
- Background color: Light beige (#F8F0E3) for a neutral, earthy backdrop.
- Accent color: Muted olive green (#81B29A) for a natural, calming effect.
- Body and headline font: 'Alegreya', a humanist serif for a friendly, approachable aesthetic.
- Use simple, line-based icons representing different moods, cuisines, and dietary preferences.
- A clean, card-based layout for displaying restaurant recommendations with clear headings and descriptions.
- Subtle transition effects and animations to enhance user interaction when filtering and exploring recommendations.
